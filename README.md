# AI Quiz System

## 📌 Project Overview

AI Quiz System is a web-based quiz application developed using Java, JSP, Servlets, JDBC, MySQL, HTML, CSS, and JavaScript.

It allows users to register, login, attend quizzes, and view their results through an interactive dashboard.

---

## 🚀 Features

- User Registration
- User Login
- Session Management
- Logout Functionality
- Java Quiz
- Python Quiz
- HTML Quiz
- Coding Quiz Modules
- Result Calculation
- Responsive Dashboard
- Database Integration

---

## 🛠 Technologies Used

- Java
- JSP
- Servlets
- JDBC
- MySQL
- HTML
- CSS
- JavaScript
- Apache Tomcat
